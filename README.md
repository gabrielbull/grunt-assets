# grunt-assets

[![Build Status](https://travis-ci.org/gabrielbull/grunt-assets.svg)](https://travis-ci.org/gabrielbull/grunt-assets)

## Getting Started

Example config

```js
grunt.initConfig({
    assets: {
        options: {
            repository: "git@github.com:myaccount/my-assets-repo.git",
            dest: "../my-assets-repo"
        }
    }
});
```
